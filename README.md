# Awesome-book

> This is a project where we had to add books, to be more specific we had to add the title of the book and the name of the author, at the moment we just created the function to add and remove the book, basic styles but it will has more eventually

## Getting Started

To get a local copy up and running follow these simple example steps.
Copy the SSH code to download all the files.

### Prerequisites

- Github.
- Code editor.
- Git Bash
- Node Js

#### Setup

Open Git Bash and run
`git clone https://github.com/desorgui/Awesome-book.git`

#### Install

Run `npm install` in the folder **_Awesome-book_**

#### Test

Run 
`npx stylelint "**/*.{css,scss}"` to test css Linter
`npx eslint .` to test Javascript Linter
`npx hint .` to test html Linter

#### Usage

Open The folder in your file explorer then, right click on index.html and open with your browser.
